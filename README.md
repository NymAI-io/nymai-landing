The official landing page for NymAI - an AI-powered engine that instantly verifies the authenticity and credibility of any text, image, or video.

## Quick Start

```bash
# Install dependencies
npm install

# Set up environment variables (optional)
cp .env.example .env
# Edit .env if needed (defaults provided)

# Start development server
npm run dev
```

Visit `http://localhost:5173` to view the landing page.
